# techflow-task-manager
Sistema de gerenciamento de tarefas com Flask e metodologias ágeis

Sistema web de gerenciamento de tarefas desenvolvido em Python utilizando o framework Flask. O projeto foi criado para aplicar conceitos de Engenharia de Software, desenvolvimento web, controle de versões com Git/GitHub e metodologia ágil Kanban.

## 📋 Descrição

O TechFlow Task Manager permite ao usuário gerenciar tarefas de forma simples e intuitiva, oferecendo operações básicas de CRUD (Create, Read, Update e Delete).

O sistema possui interface web desenvolvida em HTML e CSS e utiliza Flask como framework principal.

---

## 🚀 Funcionalidades

- Criar novas tarefas
- Listar todas as tarefas
- Editar tarefas existentes
- Excluir tarefas
- Interface web simples e intuitiva

## ▶️ Como Executar o Projeto

### 1. Clone o repositório

```bash
git clone https://github.com/SEU-USUARIO/techflow-task-manager.git
```

### 2. Acesse a pasta

```bash
cd techflow-task-manager
```

### 3. Crie um ambiente virtual (opcional)

Windows

```bash
python -m venv venv
venv\Scripts\activate
```

Linux/Mac

```bash
python3 -m venv venv
source venv/bin/activate
```

### 4. Instale as dependências

```bash
pip install -r requirements.txt
```

### 5. Execute a aplicação

```bash
python src/app.py
```

Depois abra o navegador em:

```
http://127.0.0.1:5000
```

## 📌 Metodologia Ágil

Durante o desenvolvimento foi utilizada a metodologia Kanban por meio do GitHub Projects.

As tarefas foram organizadas em três colunas:

- To Do
- In Progress
- Done

Essa organização facilitou o acompanhamento do progresso do projeto.
